# Signal Garden

Type a thought and press Enter. Each word becomes a persistent species in a small
particle ecology:

- repeated words reinforce their species;
- similar words attract one another;
- punctuation changes energy and rhythm;
- every signal loops forever through a deterministic bloom, drift, and fade cycle;
- new words expand the ecology without changing older signals' loop patterns;
- the same seed and phrase always produce the same garden;
- everything stays in the browser.

Controls:

- `Enter` — plant the current phrase
- `Escape` or click the garden — leave typing mode so shortcuts become active
- `Space` — pause/resume
- `R` — reseed and replay
- `C` — clear
- `S` — toggle sound
- `E` — export the garden as JSON
- drop an exported JSON file onto the page to restore it

Run with `python3 -m http.server 8765`, or open `index.html` directly.
