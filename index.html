05 / DESKTOP FIELD STUDY

Bring the sky home.

Native packages for Windows and macOS are on the flight path.

FABLE · WEBGPU / ELECTROBUN

MURMURATION

16,384 starlings · one peregrine · dusk becoming night
autonomous flight gpu compute

FABLE · JULY 2026

A WebGPU flocking study over procedural water. Thousands of starlings share panic, speed, color, and wingbeats while a steerable falcon turns collective motion into calligraphy.

WINDOWS

Windows 11 · current WebView2 · discrete GPU recommended

macOS

macOS 14+ · current WebKit/WebGPU · source run is experimental

Run in Windows PowerShell—not WSL—or macOS Terminal. The first launch downloads roughly 50 MB of Electrobun runtime files.

RUN FROM SOURCE

git clone https://github.com/kortexa-ai/ai-playground.git
cd ai-playground/murmuration
bun install
bun start
install Bun ↗ read the field notes ↗

Signed one-click packages for Windows and macOS are coming later. For now, the source build is the honest route.

06 / DESKTOP FIELD STUDY

Teach the sea to see.

Feeds on webcams, dropped films, or its own dreams.

FABLE · WEBGPU / ELECTROBUN

PHOTOPHORE

262,144 motes · the camera as an invisible sea · pointillism that breathes
autonomous reverie gpu compute

FABLE · JULY 2026

A quarter-million particles render whatever the machine can see—webcam, a video dropped into a folder, or a procedural dream—as living pointillism. Colors are chased, motion scatters, and every mote drifts home on a soft spring, so the picture is always present and never still.

WINDOWS

Windows 11 · current WebView2 · discrete GPU recommended

macOS

macOS 14+ · current WebKit/WebGPU · source run is experimental

Change what it dreams about: drop any video into photophore/media/. A CC-licensed jellyfish loop ships as the default film.

RUN FROM SOURCE

git clone https://github.com/kortexa-ai/ai-playground.git
cd ai-playground/photophore
bun install
bun start
install Bun ↗ read the field notes ↗

Sibling of Murmuration—same runtime, same honest source route, a different way of looking.
